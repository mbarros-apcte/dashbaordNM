# Description

Dashboard Project


## Installation

#### Virtual Environment

##### Create Virtual Enviroment
```command line
python -m venv .[name of enviroment]
```

##### Activate Virtual Enviroment
```command line
.[name of enviroment]\Scripts\activate
```

#### Install Libraries
```command line
python install -r requirements.txt
```


## Use

Run Application
```command line
flask run
```
